# Toxicology and migration

:::note Status
**Stub** — content to be written.
:::

**Covers:** Ranked risk framework for degradation products and migrating substances from homebrewing materials. What is actually dangerous, what is theoretical, and what is negligible.

**Risks to rank (highest to lowest):**
1. NR/SBR elastomers — N-nitrosamine precursors. Invisible to inspection. Only safeguard is material selection. Non-negotiable replacement recommendation. See [Natural rubber](./05-materials-register/12-nr.md).
2. POM (acetal) — formaldehyde migration under degradation. Largely replaced in DuoTight, but still present in John Guest fittings. See [POM](./05-materials-register/08-pom.md).
3. PC (polycarbonate) — BPA migration. Largely eliminated from current equipment. See [PC](./05-materials-register/06-pc.md).
4. PS (polystyrene) — styrene migration. Airlock context; GPPS vs. SAN distinction. See [PS/SAN](./05-materials-register/05-ps-san.md).
5. DDBSA in beer from ABNS residue — worst case 0.21 ppm, well below flavour or safety thresholds. See [The wet-dry cycle model](./04-wdc-model.md).
6. Microplastics — sources predominantly environmental; glass bottles measured higher than PET in ANSES 2025 data. See [Microplastics](./07-microplastics.md).

**Key principle:** Visible failure before toxicology — in most cases, material degradation is visible (cracking, swelling, discolouration) before toxic migration becomes significant. NR/SBR is the critical exception: nitrosamine precursor release is chemically invisible and inspection provides no reassurance.

**Regulatory framework:** [EU Regulation 1935/2004](https://eur-lex.europa.eu/legal-content/EN/TXT/?uri=CELEX:02004R1935-20090807) (food contact general), [EU Regulation 10/2011](https://eur-lex.europa.eu/legal-content/EN/TXT/?uri=CELEX:02011R0010-20230923) (plastics specific), IARC classifications where relevant.
